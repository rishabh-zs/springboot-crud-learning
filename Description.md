# Assignment11 - Repository Description

`Assignment11` is a Spring Boot-based CRUD learning project that demonstrates RESTful API design, layered backend architecture, and practical production concepts. It includes authentication and basic security, Redis-backed caching for better performance, transaction management for data consistency, and PostgreSQL-based persistence. The project also integrates Zipkin for distributed tracing, includes unit and integration tests with JUnit and Mockito, and is containerized with Docker for easy local setup and deployment.

## Highlights

- A Spring Boot-based CRUD application built for learning and practicing backend development concepts.
- Demonstrates RESTful API design and implementation.
- Implements authentication and basic security mechanisms.
- Uses caching (Redis) to improve performance.
- Showcases transaction management for data consistency.
- Includes unit and integration testing using JUnit and Mockito.
- Integrates Zipkin for distributed tracing and request monitoring.
- Uses PostgreSQL (or any relational DB) for data persistence.
- Containerized using Docker for easy setup and deployment.
- Covers real-world backend development practices and architecture.
